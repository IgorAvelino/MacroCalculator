## _Instalation_
- First of all, you need [Python3](https://www.python.org/downloads/)
- After this, clone the repository:
```bash
  git clone https://github.com/IgorAvelino/MacrosCalculator
```
```bash
  cd MacrosCalculator
```
- And just execute the python file:
```bash
    python3 app.py
```
- or:
```bash
    py app.py
```
